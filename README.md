# Crypto Trading GRSM Package

**Global Risk & State Manager (GRSM)**  
Core risk engine for multi-strategy trading with:

- Shadow ledger
- Risk rules engine
- Event bus
- 3-step order validation
- Fail-safe and emergency mechanisms
- Deterministic replay

## Structure

- core/risk: GRSM core modules  
- core/strategies: base strategy templates  
- tests: unit and integration tests
